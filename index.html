CSS - Cascading Style Sheet

O CSS é um folha de estilo em cascata que segue o famoso ditado: os últimos seram os primeiros, ou seja, se tivermos mudanças de estilos com a mesma propriedade e diferentes valores fica o que estiver por último. Pórem, existem varidas formas de prioridades que escreveremos a seguir:

Existem três forma de aplicar o CSS:

  • Inline: Na mesma linha da Tag.
  • Internal: Com a tag Style dentro da tag Head.
  • External: Com um arquivo externo .css - Precisa ser linkado com a folha.

Seletores em CSS

Os seletores em CSS servem para conseguirmos definir as regras de estilo para o seletor escolhido. Através dele selecionamos e estilizamos.É o link que fazemos entre o elemento HTML e a sua estilização com CSS.

Existem diversos tipos de seletores em CSS:

  • Class - Para selecionarmos usamos o . seguido de sua Class.
  • ID - Para selecionarmos usamor o # seguido de seu ID
  • Universais - Para selecionarmos usamos o *
  • Genéricos - Para selecionarmos usamos alguma referência genérica.

Pseudo-Classes em CSS

São formas de relações entre seletores que possuem objetivos específicos.
1 - Ajudam a diminuir o número de classes criando uma maior flexibilidade.
2 - São usadas para definir ações.

  • :last-child - Todos os últimos filhos seram estilizados.
  • :only-child - Todos os filhos seram estilizados
  • :invalid - Todas as tags form/fildset/input que não forem satisfeitas em suas condições seram estilizadas para demarcar isso.
  • :hover - Todas as vezes que passarmos o mouse serão aplicadas determinadas estilizações.
  • :focus - Todas as vezes que determinada Tag for selecionada pela tecla TAB serão aplicadas determinadas estilizações.
  • :visited - Indica se o link foi visitado.
  • :active - Usado para selecionar e estilizar links ativos

Pseudo-Elementos em CSS

São formas de relações entre seletores que possuem objetivos específicos.
* A diferença comparado ao Pseudo-Elementos (Que tem o funcionamento semelhando a uma classe) é que ele funciona como uma parte a mais do código HTML.

  • ::first-line - Todas as primeiras linha serão estilizadas.
  • ::first-letter - Todas as primeiras letras serão estilizadas.
  • ::before - Usado estilizar alguma coisa depois de outra.
  • ::after - Usado para estilizar alguma coisa antes de outra.

Combinadores em CSS

São formas de combinações entre os seletores que possuem objetivos específicos.

  • Descendente - Usa a lógica parental para estilizar tags específicas.
  • Filho (>) - Todas os filhos serão alterados.
  • Adjacente (+) - Somente o irmão mais próximo será alterado.
  • General (~) - Seleciona todos os elementos que vem depois independente de suas relações parentais.

Modelo de Caixa - Box Model

  • Width - Altera horizontalmente a distância.
  • Height - Alteral verticalmente a distância.
  • Padding - Altera a distância entre o conteúdo e a borda.
  • Border - Altera a distancia entre o padding e a margem.
  • Margim - Altera a distância entre a borda (camada mais externa) e o espaço do site.
